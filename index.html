Segurança
🔒 Firestore Rules — seus dados são acessíveis apenas pelo seu Google ID autenticado.
🔒 HTTPS — toda comunicação é criptografada (GitHub Pages força HTTPS).
🔒 CSP — Content Security Policy ativa, bloqueando scripts externos não autorizados.
🔒 Tokens — tokens Google ficam apenas em memória, nunca no localStorage.
⚠️ Ação necessária: Configure as Firestore Security Rules no console Firebase:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null
        && request.auth.uid == userId;
    }
  }
}
